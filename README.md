
SwordAdmin 0.1 Beta 2022
===============

> 基于Webman的PHP快速开发框架，让你更专注于业务代码的开发

## 主要特性
* 前后端分离
* 集成RBAC权限管理
* 基于Webman高性能框架

## 开始使用

## 安装依赖
```shell
yarn
```

### 配置

```
src/config/index.php
    > API_URL: 接口地址
```

# 启动项目
```shell
yarn dev
```
